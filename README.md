# React Basics — Projects Repository

This repository contains multiple projects developed while learning React, TypeScript, and full-stack web development.

## Repository Structure

```
React basics/
├── ecommerce-project/     → Main e-commerce frontend (React + TypeScript + Vite)
├── ecommerce-backend/     → Backend/API for the e-commerce project
├── chatbot-project/       → Chat application project
└── Old projects/          → Earlier practice projects and exercises
```

## Projects

### ecommerce-project

A full-featured e-commerce web application built with **React**, **TypeScript**, and **Vite**.

**Features:**
- Product search
- Shopping cart with live quantity tracking
- Checkout flow (order summary, delivery options)
- Order tracking with progress updates
- Client-side routing (React Router)

**Tech stack:** React 19, TypeScript, Vite, React Router, Vitest

---

### ecommerce-backend

The backend/API layer that serves data to the `ecommerce-project` frontend (product data, orders, etc.), using **Axios** for data fetching integration.

---

### chatbot-project

A simple interactive chatbot that responds to user commands. Completed features:

- **Get the current time**
- **Flip a coin**
- **Roll a dice**

---

### Old projects

A collection of earlier practice projects and exercises built while learning JavaScript and React fundamentals (not actively maintained).

Notably includes `ecommerce-project.js`, the original JavaScript version of the e-commerce project before it was converted to TypeScript. Kept here for reference; the actively maintained version is in `ecommerce-project/`.
